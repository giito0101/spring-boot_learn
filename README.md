# spring-boot learn
Sprng Bootの学習用リポジトリ

## Description
### 1_MyBootApp
Mavenプロジェクトで作成したプロジェクト  
中身は動作検証

### 2_MyBootGApp
Gradleプロジェクトで作成したプロジェクト上で  
中身は動作検証

## Usage
### Install
```sh
別記事に書く予定・・・
```

### Build
#### 1_MyBootApp
mvnプロジェクト直下に移動
```sh
mvn build
```

#### 2_MyBootGApp
Gradleプロジェクト直下に移動
```sh
gradle build
```

### Run

#### 1_MyBootApp
```sh
mvn spring-boot:run
```

#### 2_MyBootGApp
```sh
gradle bootRun
```
